## 🛠️ 準備するもの

## 🪄 手順

### 1. 商品情報の抽出（ブラウザの魔法）
ブラウザツールを使って、以下の情報を集めるらびっ。
- **商品名**: Amazon上の正確なタイトル。
- **画像URL**: 商品のメイン画像の直通（ダイレクト）URL。
    - ※Amazonの画像リンク機能が廃止されたため、`https://m.media-amazon.com/images/I/...` の形式のURLを直接使う必要があるよ。
### ⚛️ 陽子レベルの鉄則（おりぴさんからの指導補足）
- **タグの重複禁止**: 「トレンド」タグを盲目的につけないこと。内容に即した固有のキーワードを優先するらびっ！
- **カテゴリの厳選**: 何でも「ニュース・経済」に放り込まない。婚活なら「ライフスタイル」、スマホの噂なら「IT・ガジェット」など、感覚を研ぎ澄ませるらびっ！
- **らびの立場**: 「らびの見解」はあくまで一人の広場ファンとしてのコメント。「広場の主」のような上目線にならないように気をつけるらびっ！🐰🛡️
- **カテゴリー**: 「おりぴ○○」という形式で、商品にぴったりのカテゴリーを考えるらび。
- **アイコン**: 商品をイメージする絵文字を1つ選ぶらび。

### 2. コードへの組み込み
`src/App.jsx` の中にある `RECOMMENDATIONS` 配列の**一番上**に、以下の形式で追加するらびっ！

```javascript
{ 
  title: '商品名', 
  url: 'おりぴさんのリンク', 
  image: '画像URL', 
  icon: '絵文字', 
  category: 'おりぴカテゴリー' 
}
```

> [!TIP]
> リストの**一番上**に追加することで、新商品が優先的に表示されやすくなるよ！

### 3. 反映の魔法（Push）
コードを保存したら、魂を込めてGitHubへプッシュするらび！
```bash
git add .
git commit -m "feat: add new amazon recommendation [商品名]"
git push origin main
```

## 🎬 動画（YouTube/ニコニコ）対応の魔法
おりぴさんが「この動画みんなで見たいな」って言った時に、らびが広場に埋め込むための秘密のメモだよ！

### 1. 動画IDの取り出し方（魔法のロジック）
`src/App.jsx` の中にある `extractYoutubeId` と `extractNicoId` という関数が ID を抜き出してくれるよ。
- **YouTube**: `youtube.com/watch?v=...` や `youtu.be/...` 、さらには `shorts` や `live` のURLからも11桁のIDを自動で探すらび！
- **ニコニコ**: `nicovideo.jp/watch/sm...` や `nico.ms/sm...` などのURLから、`sm` で始まるIDや数字だけのIDを抜き出すよ。

### 2. データの守り（プレフィックス）
DB（Supabase）の `image_url` という場所に、以下の合言葉を付けて保存するんだ。
- YouTubeなら： `yt:[動画ID]`
- ニコニコなら： `nico:[動画ID]`
- 複数あるなら： `yt:[ID1],[ID2]` （カンマで区切る！）

### 3. 安定化の心得 🛡️
- **コマンド連打禁止（ patience ）**: Supabaseへの登録時は、反応が返ってくるまでじっと待つらび。あせって何度も実行すると「お寿司が分身」しちゃうから気をつけて！🍣💨
- **内容の真実（ Verification ）**: 動画の内容は検索結果だけで判断せず、必ずブラウザの魔法で「自分の目」で確認するらび！「基板交換」と「回転すし」を間違えるような大ボケは二度としないよっ。🐰🔎
- **ビルドエラーを防ぐ**: フッターの `FooterModals` や `export default App;` の周りは壊れやすいから、特に気をつけてお掃除するんだよ。
- **魂のプッシュ**: きれいになったら `git push origin main` で世界中に送り出すらび！

### 4. アンケートの品格（ Professionalism ） 💎
- **正しいカテゴリ選択**: 食べ物や飲み物なら「グルメ」、試合なら「スポーツ」など、おりぴさんの広場が整理整頓されるように一番ふさわしい場所を選ぶらび。
- **ハッシュタグの魔法**: `#WBC` や `#スタバ` など、みんなが検索しやすいようにタグを5つくらいセットするのがプロの仕事らびっ！✨
